Input Text

O input é usado para inserir uma caixa para que o usuário digite alguma informação e posteriormente, envie essa informação para o backend e/ou frontend.

Existe alguns tipos de input, aqui vamos ver o input text, que é usado para inserir texto.

Sua anatômia é bem simples: Digitando input:text o emmet já completa a TAG correta e insere também os principais atributos.



ATRIBUTO: type

Indica o tipo do input. Seu valor padrão é o próprio text, ou seja, se o atributo não for definido, automaticamente o HTML reconhece o input como text.

ATRIBUTO: name

É usado para que o backend ou frontend, busquem essas iformações digitadas pelo usuário.

ATRIBUTO: id

É um identificador, usado para vincular o input com o label

ATRIBUTO: value

É uma informação pré definida no HTML, usada para ajudar o usuário a identificar o que deve ser digitado no input. Se o usuário não digitar nada no input, o valor pré definido no HTML, será enviado para o backend e/ou frontend.

Obs.: o valor do value sobrescreve o valor do placeholder e as informações inseridas pelo usuário, sobrescreverão, os valores do value e do placeholder.

Label

Além de ser usado para víncular com o input, o label é usado para colocar uma descrição na frente do input.
